# Sistema Front-end de Cadastro de Usuários

Projeto Front-end desenvolvido com HTML, CSS e JavaScript, com o objetivo de praticar conceitos básicos de desenvolvimento web e simular um sistema de cadastro e login de usuários.

---

## 🚀 Funcionalidades

- Cadastro de usuários
- Login com validação
- Armazenamento de dados no localStorage
- Dashboard pós-login
- Logout do sistema
- Layout simples e responsivo

---

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (DOM, eventos e localStorage)
